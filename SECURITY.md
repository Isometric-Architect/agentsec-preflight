# Security Policy

## Supported Versions

This preview package is reviewed for the current main branch or preview package only. It is not production-ready.

## Reporting a Vulnerability

Please do not open public issues with exploit details, tokens, credentials, private keys, or real customer data.

For the future public repo, use GitHub Private Vulnerability Reporting if it is enabled. If private reporting is not enabled yet, open a non-sensitive issue asking maintainers for the preferred private reporting channel.

## Scope

AgentSec Preflight produces local preflight reports. It is not a security certification, not a complete security audit, and not action authorization.

Do not include real credentials, private keys, OAuth tokens, private runtime logs, customer data, or private negative-control bodies in reports or examples.
