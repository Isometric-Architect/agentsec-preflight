# Scope Boundary

AgentSec Preflight produces local preflight reports. Final deployment, security review, legal review, and action authorization stay with your team and approval workflow.

This standalone pack does not create a separate public repo, publish a release, upload to PyPI, publish to an MCP Registry, publish to Hugging Face, deploy a site, change DNS, install OpenClaw or Hermes, run a live agent, call a live tool, or perform external action.
