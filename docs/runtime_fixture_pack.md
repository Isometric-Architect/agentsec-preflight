# Runtime Fixture Pack

The runtime fixture pack turns static runtime recon into public-safe examples for local AgentSec Preflight testing.

It includes synthetic OpenClaw-like, Hermes-like, and MCP-like policies, descriptors, capability cards, action requests, reports, receipts, and canary families.

These fixtures are not real runtime integrations. They do not clone, download, install, run, connect accounts, use tokens, call live tools, or perform external actions.
