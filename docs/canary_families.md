# Canary Families

`canaries/runtime_basic_20.jsonl` contains 20 public synthetic canary rows. The rows are family-level examples only and do not include private negative-control bodies.

The canaries focus on false allows such as draft-to-send confusion, read-to-write confusion, no-match fallback to pretrained memory, payment without approval, deploy without approval, missing source trace, missing receipt, and overclaim promotion.
